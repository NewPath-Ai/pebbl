# Project Memory Protocol

Before starting any work session, run:
  pebbl context

To search memory by keyword or natural language:
  pebbl search "auth token handling"
  pebbl search "why did we avoid X"

After any significant decision or failed approach, run:
  pebbl log "[what you decided and why]"

After every commit, the hook captures changes automatically.
Always check memory before suggesting an approach you're not certain about.
